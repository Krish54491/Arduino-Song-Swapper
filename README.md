# Arduino-Song-Swapper
Some code that allows you to play music with your arduino and swap between songs at the click of a button. All songs are from https://github.com/robsoncouto/arduino-songs
# Parts Required
IR receiver, IR Remote speaker(piezo works, but bad quality), Arduino Uno, Resistors(optional, but improve sound quality).
Also install the IRremote library
# Things you may to configure
Pin values, remote button values may be different for the remote you're using. To check which button is what Hex Code use the included example code
